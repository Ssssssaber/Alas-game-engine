# Alas-game-engine
Small 2d engine using OpenGL for rendering

# Build from source
#### Windows
```console
    git clone https://github.com/Ssssssaber/Alas-game-engine.git
    cd .\Alas-game-engine\
    git submodule update --init
    ./install.bat
```
#### Linux
```console
    git clone https://github.com/Ssssssaber/Alas-game-engine.git
    cd Alas-game-engine/
    git submodule update --init
    ./install.sh
```
